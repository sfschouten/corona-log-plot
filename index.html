This is a chart to compare the number of cumulative cases over time for different countries. On the x-axis we have the number of days since a threshold (100 by default) of cases has been crossed. This threshold can be adjusted by the 'min_cases' url parameter (100, 200, 300).

We find the point at which a country crossed the threshold by interpolation. Because this is a logarithmic plot, we linearly interpolate between the log of the two datapoints closest to threshold. Countries where the amount of cases has not yet exceeded the threshold are omitted.

The information in this chart is from the dataset maintained by the Johns Hopkins CSSE; unfortunately something went wrong with March 12th's update of the dataset, for a number of countries the number of cases did not get updated and therefore repeats the number of cases from March 11th.

I'll try and continue to improve this chart, for example right now it does not include country totals; only the subdivisions are shown. Also if you want to be able to compare to Hubei (China), you'll need to set the min_cases to 444.
